# 🚀 Release 01 — Mapa L.I.L.A.S

**Data:** Junho de 2026  

---

## ✅ O que foi entregue

### 🎨 Frontend
- **Landing Page (Início):** Página inicial com hero, cards de estatísticas, seção "Sobre" e cards de funcionalidades
- **Página de Projetos de Lei:** Listagem de proposições com filtros por partido, UF, status e ano, busca por palavra-chave, paginação e ordenação

### 🗄️ Banco de Dados
- Modelagem e criação das tabelas do banco PostgreSQL
- Integração com a API da Câmara dos Deputados e do Senado Federal
- Coleta e armazenamento de proposições legislativas relacionadas ao feminicídio

### 📐 Arquitetura
- Definição da arquitetura técnica (FastAPI + React + PostgreSQL)
- Modelagem C4 do sistema
- Configuração do ambiente com Docker

---

## 🔧 O que falta para a próxima release

- Página de Gráficos (dashboards interativos)
- Página de Detalhamento do Projeto de Lei
- Integração dos valores estatísticos da landing page com a API
- Filtros adicionais e melhorias na busca

---

## 🛠️ Tecnologias utilizadas

- **Frontend:** React + TailwindCSS
- **Backend:** Python + FastAPI
- **Banco de Dados:** PostgreSQL
- **Infraestrutura:** Docker + GitHub Actions
